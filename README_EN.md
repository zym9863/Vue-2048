# Vue 2048 Game

English | [中文](./README.md)

A classic 2048 puzzle game built with Vue 3 + TypeScript + Vite.

## Features

- 🎮 Classic 2048 gameplay
- 🚀 Vue 3 Composition API
- 📝 TypeScript support
- ⚡ Vite fast build
- 📱 Responsive design

## Development Environment

- Node.js 16+
- pnpm (recommended) or npm

## Quick Start

### Install Dependencies
```bash
pnpm install
```

### Development Mode
```bash
pnpm dev
```

### Build for Production
```bash
pnpm build
```

### Preview Build
```bash
pnpm preview
```

## Project Structure

```
src/
├── components/
│   └── Game2048.vue    # Main game component
├── game/
│   └── core.ts         # Game core logic
├── App.vue             # Root component
├── main.ts             # Application entry
└── style.css           # Global styles
```
